# Gruppe11

## Denne CRUD-applikasjonen er avsluttende hjemmeeksamen i faget TSD2490 - Programvareutvikling ved Universitetet i Sørøst-Norge.

### Oppbygningen av oppgaven er som følger.
Da eksempelkoden fra deloppgavene er på engelsk, har gruppen valgt og ikke oversette eksempelkoden til norsk, men har valgt å bruke norsk der det er spesifisert i oppgaven.

Nedenfor beskrives noen utvalgte funksjonaliteter for applikasjonen.

### Deloppgaver – Fetch data
* CRUD funksjonalitet.
* Dynamisk tittel på modal-popup.
* Input validering med valideringsbeskjeder ved feilbruk.

### Ekstraoppgaver – Observasjoner
* Service for uthenting av de siste syv dagene mot Frost API-et.
* Service for lagring og uthenting av værdata i database.
* Mulighet for filtrering av visning av værdata.
